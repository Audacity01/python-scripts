# Python Scripts

A collection of handy Python scripts for everyday tasks. No heavy dependencies, just practical tools.

## Scripts

| Script | Description |
|--------|------------|
| `file_organizer.py` | Organizes files in a directory by extension |
| `json_flattener.py` | Flattens nested JSON into a flat dict |
| `password_generator.py` | Generates strong random passwords |
| `csv_to_json.py` | Converts CSV files to JSON |
| `duplicate_finder.py` | Finds duplicate files by content hash |

## Usage

Each script is standalone. Run with:
```bash
python scripts/file_organizer.py /path/to/messy/folder
```
